This is a simple workflow engine for ColdFusion. Combined with Coldspring this can work quite 
nicely to define your application workflows. This was inspired by Dan Skagg's CFObjective 
presentation and a JavaWorld article on Spring Workflows. The following are the links.

http://dan.skaggsfamily.ws/2010/04/24/building-advanced-workflows-with-coldspring-at-cf-obective/
http://www.javaworld.com/javaworld/jw-04-2005/jw-0411-spring.html
